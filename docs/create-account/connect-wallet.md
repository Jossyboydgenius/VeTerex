## Troubleshooting

<details>

<summary>"Invalid Domain" Error</summary>

If using the Chrome extension, the extension domain may need to be registered in Wepin Workspace. Contact support if you encounter this.

</details>

<details>

<summary>Can't See Wallet</summary>

* Refresh the page/extension
* Try logging out and back in
* Check your internet connection

</details>

<details>

<summary>Wrong Network</summary>

VeTerex only works on Verychain. Ensure you're connected to the correct network.

</details>
