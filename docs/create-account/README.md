# create account

Setting up your VeTerex account is quick and easy. Here's everything you need to get started.

## Account Creation

When you connect your wallet for the first time, your VeTerex account is automatically created!

### What Gets Created

| Component          | Description                             |
| ------------------ | --------------------------------------- |
| **Wallet Address** | Your unique identifier on Verychain     |
| **Profile**        | Your VeTerex user profile               |
| **Collection**     | Empty NFT collection ready to be filled |
