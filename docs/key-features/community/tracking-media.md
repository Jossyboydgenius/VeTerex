## Completion Requirements

Media is considered "complete" when:

| Type               | Requirement         |
| ------------------ | ------------------- |
| **Movies**         | \~95% watched       |
| **Episodes**       | Full episode viewed |
| **Books**          | Marked as complete  |
| **Manga Chapters** | Full chapter read   |

## Privacy & Control

* **Toggle anytime** - Turn tracking on/off in Settings
* **No background tracking** - Only tracks active sessions
* **Your data** - Stored locally until completion
