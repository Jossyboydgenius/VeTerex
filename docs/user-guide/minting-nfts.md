## NFT Properties

Each NFT contains:

| Property | Description |
|----------|-------------|
| **Token ID** | Unique identifier |
| **Media ID** | Reference to the media |
| **Media Type** | Book, Movie, Anime, etc. |
| **Media Name** | Title of the content |
| **URI** | Metadata/image reference |

## Soulbound = Non-Transferable

Your NFTs are **permanently bound** to your wallet:

- Cannot be sold
- Cannot be transferred
- Cannot be traded

This ensures authenticity - your achievements are truly yours!

## After Minting

Once you have an NFT:

### Join Community Groups

### Find Similar Users
Discover people with matching achievements.

### Build Your Collection
All NFTs appear in your profile collection.

## Burning NFTs

If you want to remove an achievement:

1. Go to your Collection
2. Select the NFT
3. Choose "Burn" option
4. Confirm destruction
