## Using the Tracker

### With Chrome Extension

1. **Install Extension**
   - Load the VeTerex extension in Chrome

2. **Enable Tracking**
   - Allow tracking when prompted, or enable in Settings

3. **Browse Normally**
   - Visit supported sites and consume media

4. **Check Progress**
   - Open the extension to see active tracking
   - View progress bars for current media

## Completion Requirements

Media is considered "complete" when:

| Type | Requirement |
|------|-------------|
| **Movies** | ~95% watched |
| **Episodes** | Full episode viewed |
| **Books** | Marked as complete |
| **Manga Chapters** | Full chapter read |

## Privacy & Control

- **Toggle anytime** - Turn tracking on/off in Settings
- **No background tracking** - Only tracks active sessions
- **Your data** - Stored locally until completion
